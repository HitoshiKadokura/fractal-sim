## Overview
The Chaos Sphere Model uses a fractal field (\( \Phi_{\text{fractal}}, D_f = 1.80 \)) to predict:
- CMB power spectrum (99.96% consistency with Planck 2018)
- Non-Gaussianity (\( f_{\mathrm{NL}} = 3.0 \pm 0.5 \))
- Dark matter clustering (\( S_8 = 0.76 \pm 0.02 \))

## Contents
- `src/fractal_sim.py`: Monte Carlo simulation code
- `data/sim_results.csv`: Simulation outputs
- `docs/methodology.md`: Theoretical background
- `requirements.txt`: Dependencies
